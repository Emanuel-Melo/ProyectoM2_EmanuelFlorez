# 🚀 MiniBlog API

API REST desarrollada con **Node.js + Express + PostgreSQL** para gestionar **authors, posts y comments**.

Este proyecto simula el backend de una plataforma tipo blog, permitiendo operaciones CRUD completas y sirviendo como base para futuras integraciones (frontend, autenticación, etc.).

---

## 📌 Características

- CRUD completo de **authors**
- CRUD completo de **posts**
- CRUD de **comments**
- Relación 1:N → authors → posts → comments
- Validaciones y manejo de errores
- Persistencia en PostgreSQL
- Tests automatizados con Jest + Supertest
- Arquitectura modular

---

## 🧱 Tecnologías
